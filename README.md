# pewter - UI Colors from Images

> NOTE: Pewter is not finished and is still a work in progress!

Pewter is a Node.js pacakge which gathers the dominant colors from an image which can be used for your UI. An example use case for might be for a music player whose colors should match the colors of the album art for the song being played.

## Installation

```bash
npm install pewter
```

## Running the Demo

```bash
npm run demo-serer
```

Visit `http://localhost:8080`.
